# How to Make Your Job as ASN More Easy

[![MIT License](https://img.shields.io/badge/license-MIT-blue.svg)](LICENSE)

JavaScript bookmarklets to automate e-Kinerja tasks.

## How to Use

1. Open the e-Kinerja page you want to automate.
2. Open browser developer tools:
   - Press **F12** on your keyboard.
   - If F12 does not work, press **Ctrl + Shift + I**.
3. Go to the **Console** tab.
4. If this is your first time using the console, type **"allow pasting"** and press Enter to enable pasting.
5. Open the script file you need from the `src` folder.
6. Copy the entire code.
7. Paste it into the browser console and press Enter.

## Scripts

All files are in the `src` folder.

| File | Description |
|------|-------------|
| hapus-semua-penilaian.js | Hapus semua penilaian |
| hapus-rencana-aksi.js | Hapus rencana aksi |
| isi-penilaian.js | Isi penilaian |
| realisasi.js | Isi realisasi |
| rencana-aksi.js | Isi rencana aksi |
